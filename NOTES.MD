# Method Safety and Method Idempotendy

| HTTP Method    | Safe?        | Idempotent?  |
| :------------- | :----------: | :----------: |
|  __GET__       | yes          | yes          |
| __OPTIONS__    | yes          | yes          |
|  __HEAD__      | yes          | yes          |
|  __POST__      | no           | no           |
|  __DELETE__    | no           | yes          |
|  __PUT__       | no           | yes          |
|  __PATCH__     | no           | no           |
